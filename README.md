# 📱 CustomerApp

CustomerApp is an Android application built with **Kotlin & Java** in **Android Studio** for managing customer data.  
The app allows users to **add, view, search, and manage customers** with a clean and simple interface.

## 🚀 Features
- Add new customers (Name, Age, Active status)
- View all customers
- Search customers by name
- Simple and intuitive UI

## 🛠 Tech Stack
- **Kotlin & Java** – Core app development
- **Android Studio** – IDE
- **XML** – UI design
